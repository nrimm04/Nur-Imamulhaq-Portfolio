# 🌐 MY PORTFOLIO- Welcome to My World

Welcome to the repository of my personal website! This project is a simple, clean, and responsive digital biodata that showcases my life, education, hobbies, and achievements.

👉 **[Live Demo Here](https://nrimm04.github.io/MY_PORTFOLIO/)**

---

## 👨‍💻 About the Project
This website was built to introduce myself to the digital world. It contains several sections including:
* **Biodata:** Personal details and introduction.
* **Hobbies:** My love for fishing since I was 7 years old.
* **Family:** A little tribute to my supportive family.
* **Education & Experience:** My journey from school to UiTM Rembau and my past work experiences.
* **Gallery:** A collection of memorable photos with family and friends (Youngblood group!).

---

## 🛠️ Built With
This website is built using pure frontend technologies without any heavy frameworks, making it fast and lightweight:
* **HTML5** - For the structure and content.
* **CSS3** - For the dark-themed styling and modern layouts (Flexbox).
* **JavaScript** - For smooth scrolling functionality.
